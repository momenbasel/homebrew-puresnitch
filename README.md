# homebrew-puresnitch

Homebrew tap for [PureSnitch](https://github.com/momenbasel/puresnitch) — free, open-source application firewall for macOS.

## Install

```bash
brew tap momenbasel/puresnitch
brew install --cask puresnitch
```

## Uninstall

```bash
brew uninstall --cask puresnitch
brew untap momenbasel/puresnitch
```
